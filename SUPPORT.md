# Support

## Getting Help

This is a personal learning repository, not a supported product. That said, here are the best ways to get help:

### If a notebook won't run
1. Make sure you have completed the [Quick Start](README.md#-quick-start) setup
2. Verify your Python version is 3.11+: `python --version`
3. Reinstall dependencies: `pip install -r requirements.txt`
4. Check [existing issues](../../issues) — your problem may already be resolved

### If you find an error in the content
Open a [bug report](../../issues/new?template=bug_report.yml).

### If you have a question about the material
Open a [GitHub Discussion](../../discussions) — questions about concepts, techniques, or implementations are welcome there.
