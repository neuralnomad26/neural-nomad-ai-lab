# Module 06 — LLM Engineering

> **Status:** ⬜ Upcoming

Building production applications with large language models. Prompt engineering, RAG, function calling, and evaluation.

---

## Learning Objectives

- Write effective prompts using systematic techniques (CoT, few-shot, ReAct)
- Build Retrieval-Augmented Generation (RAG) pipelines from scratch
- Use function calling / tool use with OpenAI and Anthropic APIs
- Evaluate LLM outputs rigorously
- Understand cost, latency, and quality tradeoffs when choosing models

---

## Topics Covered

| Notebook | Topics |

| `01_api_fundamentals.ipynb` | OpenAI & Anthropic APIs, tokens, pricing |
| `02_prompt_engineering.ipynb` | Zero-shot, few-shot, CoT, self-consistency |
| `03_function_calling.ipynb` | Tool use, structured outputs, JSON mode |
| `04_rag_fundamentals.ipynb` | Chunking, embedding, vector search, generation |
| `05_advanced_rag.ipynb` | Re-ranking, hybrid search, query expansion |

| `08_streaming_and_async.ipynb` | Streaming responses, async clients |
