# Module 05 — Natural Language Processing

> **Status:** ⬜ Upcoming

From bag-of-words to transformers. This module builds the NLP intuition needed to work effectively with LLMs in Module 06.

---

## Learning Objectives

- Process and represent text data numerically
- Understand the transformer architecture deeply (attention is all you need)
- Use HuggingFace for classification, NER, summarisation, and generation
- Work with embeddings for semantic search and similarity
- Understand how pre-training and fine-tuning work

---

## Topics Covered

| Notebook | Topics |
|----------|--------|
| `01_text_preprocessing.ipynb` | Tokenisation, stemming, lemmatisation, stopwords |
| `02_classical_nlp.ipynb` | Bag-of-words, TF-IDF, Naive Bayes classifier |
| `03_word_embeddings.ipynb` | Word2Vec, GloVe, fastText |
| `04_transformer_architecture.ipynb` | Self-attention, positional encoding, encoder/decoder |
| `05_huggingface_pipelines.ipynb` | Zero-shot classification, NER, QA, summarisation |
| `06_sentence_transformers.ipynb` | Semantic search, cosine similarity, clustering |
| `07_fine_tuning.ipynb` | Fine-tuning BERT for custom classification |
